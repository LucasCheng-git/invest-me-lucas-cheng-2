# InvestMe 🚀

A platform connecting startups and investors — built with React, Vite, and Supabase.

## Features
- 🚀 Startups can upload or record pitch videos (max 3 min)
- 💼 Investors discover pitches filtered by industry
- 💬 Real-time messaging between startups and investors
- 🔐 Supabase Auth, Database, Storage & Realtime

## Tech Stack
- React 18 + Vite
- Supabase (Auth, PostgreSQL, Storage, Realtime)
- Deployed on Vercel

## Getting Started
```bash
npm install
npm run dev
```
